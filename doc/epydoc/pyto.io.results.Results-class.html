Reads results from file, parses the data, assignes it to local arttributes and orders them according to the ids.

Columns should be set so that only the part of results that forms a table (2d array) is read.

getVersion(self, file)

source code 

Finds version of the results file.

First it reads file to find the svn revision number of the script that was used to write this results file. Then sets the version of the results file to be the highest element of self.versions that is still lower than the revision.

Requires self.versions, which is normally not specified in this class. This method should be called from instances of classes that inherit from this class and have self.versions.set (such as Connections).

Argument:

  • file: results file

Retirns: (int) version
