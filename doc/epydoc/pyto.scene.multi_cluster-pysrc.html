 20      """ 
 21      An instance of this class can contain up to six different clusterings of 
 22      the same data (see below) 
 23   
 24      Clusters can be formed in the following ways: 
 25   
 26      1) Clustering based on connectivity 
 27   
 28      All boundaries that are linked to each other via connections (connectors)  
 29      form a boundary cluster. In the same way, all connections 
 30      that are connected to each other vie boundaries form a connections 
 31      cluster. These boundary and connection clusters are dual to each other, 
 32      because one uniquely determines the other. 
 33   
 34      Clustering method: connectivity() 
 35   
 36      Attributes (pyto.segmentation.Cluster objects): 
 37        - connectivityBoundaries: clusters of boundaries based on linkage by  
 38        connectors 
 39        - connectivityConnections: clusters of connectors based on linkage by  
 40        boundaries 
 41         
 42      The above two clustering objects (connectivityBoundaries,  
 43      connectivityConnections) may contain the following attributes: 
 44        - ids: list of cluster ids 
 45        - clusters: list corresponding to ids where each element is a set that 
 46        contains ids of boundaries or segments belonging to that cluster 
 47        - nItems: (list corresponding to ids) number of items in each cluster, 
 48        where items are boundaries in clusters of boundaries and connections in  
 49        clusters of connections. 
 50        - nConnections: (list corresponding to ids) number of connections in each 
 51        connection cluster 
 52        - nLinks: (list corresponding to ids) number of links in each 
 53        connection cluster (links are connections where multiple connections  
 54        betweeen the same boundaries are not taken into account) 
 55        - euler, nLoops: (list corresponding to ids) euler number and the number  
 56        of independent loops formed by boundaries and connections of each cluster 
 57        - eulerLinks, nLoopsLinks: the same as euler and nLoops, but based on  
 58        boundaries and links (that is neglecting multiple connections betweeen 
 59        the same boundaries)       
 60        - dataIds: list of data (boundary or connector) ids 
 61        - nClusters: number of clusters 
 62   
 63      2) Hierarchical (distance based) clustering 
 64   
 65      Hierarchical clustering is applied separately to boundaries and connections 
 66      to yeild hierarchy boundary and hierarchy connection clusters, respectively. 
 67      Dual clusters to these can be obtained, however hierarchical connection 
 68      (boundary) clusters do not have to be identical to dual hierarchy  
 69      connection (boundary) clusters (obtained from hierarchical boundary/ 
 70      connection clusters).   
 71   
 72      Clustering methods: 
 73        - hierarchicalBoundaries(): hierarchical clustering of boundaries 
 74        - hierarchicalConections(): hierarchical clustering of connections 
 75   
 76      Attributes (pyto.segmentation.Cluster objects): 
 77        - hierarchyBoundaries: hierarchical clusters of boundaries 
 78        - hierarchyConnections: hierarchical clusters of connections 
 79        - dualHierarchyBoundaries: hierarchical clusters of connections dual to 
 80        hierarchyBoundaries 
 81        - dualHierarchyConnections: hierarchical clusters of boundaries dual to  
 82        hierarchyConnections 
 83   
 84      The above two clustering objects (connectivityBoundaries,  
 85      connectivityConnections) may contain the following attributes: 
 86        - ids: list of cluster ids 
 87        - clusters: list corresponding to ids where each element is a set that

 92        - dataIds: list of data (boundary or connector) ids 
 93        - nClusters: number of clusters 
 94
