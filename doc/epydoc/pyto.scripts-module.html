Submodules [hide private]
  • pyto.scripts.add_segments: Adds segments form different segment (labels) files.
  • pyto.scripts.blank: Blanks (sets to a given value) those voxels of a tomogram that correspond to given segments.
  • pyto.scripts.bulk_analysis: Batch execution of scripts.
  • pyto.scripts.classify_connections: Depreciated.
  • pyto.scripts.cleft: This script performs the following tasks: 1) Makes cleft regions and analyzes their greyscale density.
  • pyto.scripts.cleft_stats: Statstical analysis of the synaptic cleft
  • pyto.scripts.cluster: Makes and analyzes clusters of (already existing) boundaries and segments.
  • pyto.scripts.common: Contains functions often used in the scripts of this directory
  • pyto.scripts.connections: Depreciated.
  • pyto.scripts.connectors: There are two tasks preformed by this script: 1) An image is segmented using the hierarchical connectivity procedure to yield connectors (segments).
  • pyto.scripts.correlation_3d_2d: Establishes a correlation between 3D and 2D coordinate systems (e.g.
  • pyto.scripts.correlation_simple: Establishes a correlation between two arbitrary systems and correlates positions of objects of interest (targets) between these two systems.
  • pyto.scripts.correlation_simple_fileio: Establishes a correlation between two arbitrary systems and correlates positions of objects of interest (targets) between these two systems.
  • pyto.scripts.correlation_two_step_lm_em: Establishes a correlation between a light microscopy (LM) image and EM stage coordinates and correlates positions of objects of interest (targets) between EM and LM.
  • pyto.scripts.discs_to_balls: Reads a labels file containing discs and writes another labels file with the discs extending to balls having same centers and radii as the discs.
  • pyto.scripts.extract_particles: Extracts sub-images containing individual particles based on the corresponding segment positions.
  • pyto.scripts.extract_points: Extract coordinates of (some) points of one or more segments of a segmented image.
  • pyto.scripts.layers: Makes layers in a given region starting from one or between two specified boundaries.
  • pyto.scripts.neighborhood_grey: Gray value (density) analysis of parts of segments (typically layers) that are in the vicinity of some specified regions.
  • pyto.scripts.presynaptic_catalog: Segmentation and analysis meta-data
  • pyto.scripts.presynaptic_stats: Statstical analysis of the presynaptic terminal.
  • pyto.scripts.presynaptic_stats_common: Functions used to display analysys results for presynaptic_stats.py script.
  • pyto.scripts.regions: Basic greyscale density and morphology analysis of any kind of segments.
  • pyto.scripts.sort_series: Sorts series, fixes image headers, limits the data and calculates the dose.
  • pyto.scripts.tomo_info: Specifies parameters common to different individual tomogram (synapse) processing scripts in order to make setting up multiple scripts easier.
  • pyto.scripts.vesicles: Basic greyscale density and morphology analysis of vesicle-like segments.
