   1  #!/usr/bin/env python 
   2  """ 
   3  This script performs the following tasks: 
   4      1) Makes cleft regions and analyzes their greyscale density. If they  
   5  alreadyexist, cleft layers can be read. The following regions can be made  
   6  (from 0 to all 3): 
   7          a) Layers (parallel to the cleft) 
   8          b) Columns (perpendicular to the cleft, each column is trans-cleft) 
   9          c) Layers on columns 
  10      2) Hierarchical connectivity segmentation of the cleft and the 
  11  connnector analysis 
  12      3) Connector classification (optional) 
  13      4) Makes layers on segments and analyzes their greyscale density  
  14  (optional, requires layers). 
  15   
  16  The main applications of this script is to find and analyze connectors between  
  17  giventwo relatively parallel boundaries, such as the membranes of a synaptic 
  18  cleft. 
  19   
  20  Important notes: 
  21   
  22    - In order to make setting up multiple scripts easier, parameters common  
  23  to these scripts are expected to be read from tomo_info.py file. The location  
  24  of this file is given as argument path of common.__import__(). These parameters 
  25  are set in this script in lines having the following form: 
  26   
  27    if tomo_info is not None: labels_file_name = tomo_info.labels_file_name 
  28   
  29  Specifiying another value for the same parameter (here labels_file_name)  
  30  overrides the value from tomo_info. 
  31   
  32    - Incorrect id specification accounts for a majority of problems encountered  
  33  while running this script. In case an exception is thrown, please check the ids 
  34  carefully first. 
  35   
  36    - For many variables more than one option is given. Uncomment the desired one 
  37  and adjust the value if needed, and comment out the other options. 
  38   
  39    - Currently (r997) division by 0 warnings are printed. These might be ignored 
  40  because they most likely arise from calculating denisty of a 0-volume segment 
  41  and are also a consequence of the changed behavior of scipy.  
  42   
  43  This script may be placed anywhere in the directory tree. 
  44   
  45  $Id: cleft.py 1490 2018-11-08 16:11:29Z vladan $ 
  46  Author: Vladan Lucic  
  47  """
