   1  #!/usr/bin/env python 
   2  """ 
   3  There are two tasks preformed by this script: 
   4      1) An image is segmented using the hierarchical connectivity procedure  
   5  to yield connectors (segments).  
   6      2) These connectors are classified according the specified  
   7  classification criteria.  
   8  This script can be used to execute these two tasks together, or separately.  
   9  Both tasks can be followed by the analysis of the connectors.  
  10   
  11  The analysis can be done on all connectors (after the segmentation), on  
  12  connectors after classification, or both times.  
  13   
  14  The main applications of this script is to find and analyze connectors between  
  15  given boundaries, such as among vesicles and between vesicles and other  
  16  membranes. Each connector is typically required to make contacts with at least  
  17  two boundaries. 
  18   
  19  This script supersedes connections.py, as it has almost all functionality of  
  20  connections.py (except dynamic segmentation and making vesicle clusters) while  
  21  it has several software improvements. 
  22   
  23  This script may be placed anywhere in the directory tree. 
  24   
  25  Important notes: 
  26   
  27    - In order to make setting up multiple scripts easier, parameters common  
  28  to these scripts are expected to be read from tomo_info.py file. The location  
  29  of this file is given as argument path of common.__import__(). These parameters 
  30  are set in this script in lines having the following form: 
  31   
  32    if tomo_info is not None: labels_file_name = tomo_info.labels_file_name 
  33   
  34  Specifiying another value for the same parameter (here labels_file_name)  
  35  overrides the value from tomo_info. 
  36   
  37    - Incorrect id specification accounts for a majority of problems encountered  
  38  while running this script. In case an exception is thrown, please check the ids 
  39  carefully first. 
  40   
  41    - For many variables more than one option is given. Uncomment the desired one 
  42  and adjust the value if needed, and comment out the other options. 
  43   
  44    - Currently (r997) division by 0 warnings are printed. These might be ignored 
  45  because they most likely arise from calculating denisty of a 0-volume segment 
  46  and are also a consequence of the changed behavior of scipy.
