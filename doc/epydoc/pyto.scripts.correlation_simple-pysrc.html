  3   
  4  Establishes a correlation between two arbitrary systems and correlates  
  5  positions of objects of interest (targets) between these two systems.  
  6   
  7  The correlation procedure used in this script is direct, meaning there are no  
  8  intermediary systems. Also, both systems have to have the same dimensionality. 
  9   
 10  Conceptually, the correlation procedure consists of two steps: 
 11    1) Establish a correlation between the two systems (initial and final),  
 12  that is find a coordinate transformation between the two systems. 
 13    2) Correlate positions of obects of interest (targets) from one system to  
 14  the other. 
 15   
 16  Marker points are needed to determine the transformation (step 1). That is,  
 17  one needs to identify one set of features that are visible in both systems. 
 18   
 19  Requires the following coordinates: 
 20    - marker coordinates in the initial system 
 21    - corresponding marker coordinates in the final system 
 22    - (optional) target coordinates the initial and / or final systems 
 23   
 24  This is a very simple script where coordinates of markers and targets are  
 25  entered directly. The calculated transformation parameters and the coordinates  
 26  of the correlated target points are displayed on the standard output.  
 27   
 28  Standard usage: 
 29   
 30  1) Edit this file to enter the desired parameters (coordinates of markers  
 31  and objects of interest, and the transformation type) 
 32  2) Import and run this file (in IPython or another python shell): 
 33    >>> import correlation_simple 
 34    >>> correlation_simple.main() 
 35   
 36  Advanced usage: 
 37   
 38  1) Edit this file to enter the desired parameters (coordinates of markers  
 39  and objects of interest, and the transformation type) 
 40  2) Import this file and execute commands from main() one by  
 41  one. Check values of variables as needed: 
 42    >>> import correlation_simple 
 43    >>> from correlation_simple import * 
 44    >>> corr = Basic() 
 45    >>> corr.establish( 
 46          markers_1=markers_initial, markers_2=markers_final,  
 47          type_=transform_type) 
 48    >>> corr.transf_1_to_2.phiDeg 
 49    >>> corr.transf_1_to_2.gl 
 50        ... 
 51  Other attributes of corr are listed in the documentation for  
 52  pyto.correlative.Basic. If the initial and final systems are 2D, object  
 53  corr.transf_1_to_2 is an instance of pyto.geometry.Affine2D, if they are 3D 
 54  corr.transf_1_to_2 is an instance of pyto.geometry.AffinewD and in other cases 
 55  of pyto.geometry.Affine. Please check docs for these classes for other  
 56  attributes.
