  3   
  4  Establishes a correlation between two arbitrary systems and correlates  
  5  positions of objects of interest (targets) between these two systems.  
  6   
  7  The correlation procedure used in this script is direct, meaning there are no  
  8  intermediary systems. Also, both systems have to have the same dimensionality. 
  9   
 10  Conceptually, the correlation procedure consists of two steps: 
 11    1) Establish a correlation between the two systems (initial and final),  
 12  that is find a coordinate transformation between the two systems. 
 13    2) Correlate positions of obects of interest from one system to the other. 
 14   
 15  Marker points are needed to determine the transformation  (step 1). That is,  
 16  one needs to identify one set of features that are visible in both systems. 
 17   
 18  Requires the following coordinates: 
 19    - marker coordinates in the initial system 
 20    - corresponding marker coordinates in the final system 
 21    - (optional) target coordinates the initial and / or final systems 
 22   
 23  In this script, coordinates of markers and targets are specified by a file.  
 24  Such a file can be generated by clicking on targets in ImageJ. Other formats 
 25  can be used as long as the data is in a table format where rows correspond to  
 26  different markers and targets and the x, y, ... coordinates are specified by  
 27  columns. The calculated transformation parameters and the coordinates of the  
 28  correlated target points are written in a results file.  
 29   
 30  Standard usage: 
 31   
 32  1) Edit this file to enter the desired parameters (coordinates of markers  
 33  and objects of interest, and the transformation type) 
 34  2) Import and run this file (in IPython or another python shell): 
 35    >>> import correlation_simple_fileio 
 36    >>> correlation_simple_fileio.main() 
 37   
 38  Advanced usage: 
 39   
 40  1) Edit this file to enter the desired parameters (coordinates of markers  
 41  and objects of interest, and the transformation type) 
 42  2) Import this file and execute commands from main() one by one. Check values  
 43  of variables as needed: 
 44    >>> import correlation_simple_fileio 
 45    >>> from correlation_simple_fileio import * 
 46    >>> markers = { 
 47          'markers_1' : (coordinates_file, marker_initial_rows), 
 48          'markers_2' : (coordinates_file, marker_final_rows)} 
 49    >>> corr = Basic() 
 50    >>> corr.readPositions( 
 51          points=markers, format_=file_format, comments=comments,  
 52          skiprows=skiprows, delimiter=delimiter, columns=columns,  
 53          indexing=indexing) 
 54    >>> corr.establish(type_=transform_type) 
 55        ... 
 56    >>> corr.transf_1_to_2.phiDeg 
 57    >>> corr.transf_1_to_2.gl 
 58        ... 
 59  Other attributes of corr are listed in the documentation for  
 60  pyto.correlative.Basic. If the initial and final systems are 2D, object  
 61  corr.transf_1_to_2 is an instance of pyto.geometry.Affine2D, if they are 3D 
 62  corr.transf_1_to_2 is an instance of pyto.geometry.AffinewD and in other cases 
 63  of pyto.geometry.Affine. Please check docs for these classes for other  
 64  attributes.
