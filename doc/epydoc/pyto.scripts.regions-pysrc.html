  1  #!/usr/bin/env python 
  2  """ 
  3  Basic greyscale density and morphology analysis of any kind of segments.  
  4  Typically used regions / boundaries that are manually / Amira segmented.  
  5   
  6  Calculates: 
  7    - greyscale density statistics 
  8    - surface (membrane) and lumen greyscale density statistics  
  9    - distance to a specified region 
 10    - center and radius (for spherical segments) 
 11   
 12  This script may be placed anywhere in the directory tree. 
 13   
 14  Important notes: 
 15   
 16      - This script is essentially the same as vesicles.py (greyscale and  
 17  morphology of vesicles), only without vesicle-specific calculations.  
 18  Consequently many variables names are derived from 'vesicle' and many comments 
 19  talk about vesicles. All these should be understood as regions. 
 20   
 21    - In order to make setting up multiple scripts easier, parameters common  
 22  to these scripts are expected to be read from tomo_info.py file. The location  
 23  of this file is given as argument path of common.__import__(). These parameters 
 24  are set in this script in lines having the following form: 
 25   
 26    if tomo_info is not None: labels_file_name = tomo_info.labels_file_name 
 27   
 28  Specifiying another value for the same parameter (here labels_file_name)  
 29  overrides the value from tomo_info. 
 30   
 31    - Incorrect id specification accounts for a majority of problems encountered  
 32  while running this script. In case an exception is thrown, please check the ids 
 33  carefully first. 
 34   
 35    - For many variables more than one option is given. Uncomment the desired one 
 36  and adjust the value if needed, and comment out the other options. 
 37   
 38    - Currently (r997) division by 0 warnings are printed. These might be ignored 
 39  because they most likely arise from calculating denisty of a 0-volume segment 
 40  and are also a consequence of the changed behavior of scipy.  
 41
