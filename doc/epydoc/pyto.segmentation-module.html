Submodules [hide private]
  • pyto.segmentation.ball: Class Ball provides methods for crating and manipulating n-dimensional balls (filled spheres), in addition to methods defined in its base class Segment.
  • pyto.segmentation.bound_distance: Contains class BoundDistance for the calculation of distances between boundaries that contact the same segment.
  • pyto.segmentation.cleft: Contains class Cleft for the analysis of a cleft-like region (a region between two roughly parallel boundaries).
  • pyto.segmentation.cluster: Contains class Cluster for analysis of clustering of segments.
  • pyto.segmentation.connected: Provides class Connected for the manipulation of segments defined on the basis of their connectivity.
  • pyto.segmentation.contact: Contains class Contact for analysis of contacts between segments (from a segmented image) and boundaries specified on the same image.
  • pyto.segmentation.density: Contains class Density for the calculation of densities of gray images along segments.
  • pyto.segmentation.distance: Contains class Distance for the calculation of distances between given segments.
  • pyto.segmentation.distance_to: Contains class DistanceTo for calculations of distance between segments of a segmented images and given region(s)
  • pyto.segmentation.features: Contains class Features that provides basic functionality for the analysis of segmented images.
  • pyto.segmentation.grey: Contains class Gray for segmentation related analysis of grayscale (3D) images.
  • pyto.segmentation.hierarchy: Provides class Hiarrchy for the analysis of multiple segementations orgainized in a hierarchy (each segmentation is a subset of the next one).
  • pyto.segmentation.labels: Contains class Labels for general manipulations of an image that contains integer values (label field).
  • pyto.segmentation.morphology: Class Morphology for calculation of morphological quantities of a segmented image(label field).
  • pyto.segmentation.plane: Class Plane provides methods for crating and manipulating n-1 dimensional planes in n dimentional space, in addition to methods defined in its base class Segment.
  • pyto.segmentation.segment: Contains class Segments for manipulation of segmented image where segments do not touch each other.
  • pyto.segmentation.shapes: Class Shapes provides methods for crating and manipulating geometrical shapes, in addition to methods defined in its base class Segment.
  • pyto.segmentation.sphere: Class Shapes provides methods for crating and manipulating spheres, in addition to methods defined in its base class Segment.
  • pyto.segmentation.statistics: Class Statistics for basic statistical analysis of labeled (segmented) data.
  • pyto.segmentation.struct_el: Generates some unusual structuring elements.
