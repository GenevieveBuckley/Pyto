chisquare_2(f_obs_1, f_obs_2, yates=False)

source code 

Calculates chi-square between two arrays of observation frequencies and returns the result.

Differs from scipy.stats.chisquare() in that this function calculated significance between two distributions and that the two distributions can have different number of data points.

Arguments:

  • f_obs_1, f_obs_2: frequencies of observations 1 and 2

Returns: chi-square, associated p-value

ttest_ind_nodata(mean_1, std_1, n_1, mean_2, std_2, n_2)

source code 

Student's t-test between two independent samples. Unlike in ttest_ind(), the samples (data) are not given, instead the basic statstical quantities (mean, standard deviation and number of measurements) are used to do the test. Returns t-value and a two-tailed confidence level.

Arguments:

  • mean_1, mean_2: means of samples 1 and 2
  • std_1, std_2: standard deviations of samples 1 and 2 calculated using n-1 degrees of freedom where n is the number of measurements in a sample
  • n_1, n_2: number of measurements Arguments can be ndarrays instead of single numbers.

Returns: (t_value, two_tail_confidence)
